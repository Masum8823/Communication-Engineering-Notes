# Equipment Identity Register (EIR)

EIR is a database that keeps track of all mobile handsets on the network using their IMEI number (International Mobile Equipment Identity).  
It was introduced so the network can identify, track, and block devices that shouldn't be allowed — like stolen phones.

---

## 📌 Important Facts:

- There is only one EIR per network (centralized)
- MSC connects to EIR using the F Interface
- VLR connects to HLR using the D Interface,
- MSC to VLR using B Interface
