# Subscriber Identity Module (SIM)

SIM is the small smart card inside the phone. It carries all the subscriber-specific information.

---

## 📌 What does SIM contain?

• IMSI (International Mobile Subscriber Identity) — your unique subscriber ID  
• A secret authentication key (Ki)  
• Other subscriber info  

---

## ⚙️ What does SIM do?

• Provides personal mobility  
• Identifies the current user  
• Takes part in security and authentication procedures  
• Stores recent location data  
• Can also store personal info like your contacts/phone directory  
• For security it uses PIN (Personal Identity Number)
