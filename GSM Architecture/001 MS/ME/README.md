# Mobile Equipment (ME)

ME is the actual phone hardware. Each ME has a unique ID called IMEI (International Mobile Equipment Identity). This is what the EIR tracks.

---

## 📌 Functions of ME:

• Radio transceiver and signal processing  
• Radio-related operations: power control, timing advance, DTX (Discontinuous Transmission), SFH (Slow Frequency Hopping)  
• Call handling  
• Man-machine interface — display, keypad, speaker, microphone
