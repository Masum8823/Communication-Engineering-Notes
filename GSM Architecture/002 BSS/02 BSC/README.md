# Base Station Controller (BSC)

BSC is the brain that controls multiple BTS towers. One BSC manages several BTSs.

---

## 📌 What does BSC handle?

• Allocation of radio channels  
• Frequency administration  
• Power and signal measurements  
• Handover between two BTSs — (if both BTSs are controlled by the same BSC, the BSC handles the handover itself called (intra-BSC handover). If they belong to different BSCs, the MSC handles it.)
