# Base Transceiver Station (BTS)

BTS is the physical tower — the antenna you see on rooftops and hills. It is the access point of your phone to the network.

---

## 📌 Key Points:

• Provides physical connection between MS and the network  
• It uses Air Interface (Um)  
• Connects to BSC using the Abis Interface  
• It maintains the Um interface and minimizing transmission problems  
• A BTS is usually placed at the center of a cell  
• Its transmitting power defines the size of the cell — more power = bigger cell area  
• Each BTS has between 1 to 16 transceivers depending on how many users are in that cell  
• Each BTS serves only a single cell  

---

## ⚙️ What does BTS technically do?

• Carries out radio communication  
• Handles speech encoding, encryption, multiplexing (TDMA), and modulation/demodulation of radio signals  
• Capable of frequency hopping  
• One BTS usually covers a 120-degree sector of an area  
• A single tower with 3 BTSs covers all 360 degrees
