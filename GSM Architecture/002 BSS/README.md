# Base Station Subsystem (BSS)

BSS is the part of the GSM network that handles all radio-related functions. It sits between your phone and the NSS (core network).

Its job is to handle all the traffic signaling between the mobile phone and the network.

---

## 📌 Components of BSS:

• BTS (Base Transceiver Station) — the actual tower  
• BSC (Base Station Controller) — the controller that manages towers  

---

## 📡 Communication:

BSS communicates with the Mobile Station (MS) using the Air Interface (Um).
