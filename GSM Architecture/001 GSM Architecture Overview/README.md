# GSM Architecture

GSM network is divided into three main parts:

## 📱 MS (Mobile Station)
- Simply the user’s mobile phone

## 📡 BSS (Base Station Subsystem)
- Handles wireless communication
- Contains:
  - **BTS (Base Transceiver Station)** → Tower
  - **BSC (Base Station Controller)** → Controls multiple BTS

## 🧠 NSS (Network & Switching Subsystem)
- The brain of the entire network
- Responsible for:
  - Call handling
  - Mobility management
  - Subscriber data management

---

## 🔗 Interfaces in GSM

- **Um (Air Interface)**  
  → Between Mobile Station (MS) and BTS (tower)

- **Abis Interface**  
  → Between BTS and BSC

- **A Interface**  
  → Between BSC and MSC

---

## 🧠 Quick Summary
GSM architecture consists of three main components: MS, BSS, and NSS.  
These components work together through different interfaces to enable communication in the network.
