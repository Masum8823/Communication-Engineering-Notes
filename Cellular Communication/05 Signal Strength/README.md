# Signal Strength

Cellular systems work using radio signals that a Mobile Station (MS) receives from a Base Station (BS). The BS sends signals in layers around it, like rings. Each ring has a different signal strength, measured in dB (for example: -60 dB, -70 dB, -80 dB).

---

## 📌 Types of Signal Strength:

i. Practical Cell  
ii. Real Cell  

---

## 📐 Concept:

In an ideal situation, each cell looks like a perfect circle, and the boundary is clear and smooth. But in real life, things like buildings, trees, land shape, and weather affect the signal. Because of this, the signal does not spread clearly, and the cell shape becomes irregular instead of perfectly circular.
