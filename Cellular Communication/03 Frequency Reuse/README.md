# Frequency Reuse

Frequency reuse means using the same frequency in different cells that are far apart so there is no interference. This enhance the bandwidth of a cell.

• The N cells that together use the full set of channels = cluster  
• A cluster can be repeated M times across the coverage area  
• Total capacity: C = MkN = MS  
• Capacity is directly proportional to M (more repetitions = more capacity)  
• Cluster size N is typically 3, 7, or 12 — smaller N means higher capacity  
• Frequency reuse factor = 1/N  

---

# Frequency Reuse – Advantages

• Solves the problem of spectral congestion (limited spectrum) and too many users in a cell area.  
• Gives high capacity even when frequency is limited  
• Same radio channels can be reused in different cells  
• A fixed number of channels can serve a very large number of users by reusing them again and again in different areas
