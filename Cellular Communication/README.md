# Quick Summary — Cellular Communication

| Term | Meaning |
|------|--------|
| Cell | Radio coverage area of one BTS |
| Cluster | Group of cells using unique frequencies |
| Frequency Reuse | Same frequency used in far-away cells |
| λ (lambda) | Average call arrival rate |
| T | Average call duration |
| a = λT | Offered traffic load (in Erlangs) |
| 1 Erlang | One channel busy for one full hour |
| B(S, a) | Blocking probability (Erlang B formula) |
| S | Number of channels in a cell |
